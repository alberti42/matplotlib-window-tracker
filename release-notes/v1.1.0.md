## Summary

This release focuses on making the package simpler, more Matplotlib-compatible, and easier to use in an IPython + script workflow.

Highlights

## Changes
- Simplified nonblocking behavior around a minimal, tested recipe (plt.pause(...)).
- Clear separation of responsibilities:
  - refresh(fig) for per-figure updates in loops
  - show(block=False) as a global "GUI tick"
  - show(block=True) as terminal-run fallback to keep windows open
- Removed backend guessing: users select backends explicitly (%matplotlib ... or matplotlib.use(...)); added recommended_backend() helper.

## Breaking changes
- Removed ensure_backend() (no automatic backend selection).
- Removed subplots() wrapper (use plt.subplots(num=..., clear=True, ...) directly).
- show() is keyword-only and no longer accepts show(fig).
- refresh(..., in_foreground=True) replaces earlier "raise window" naming.

## Tooling
- Added GitHub Actions to build and attach .tar.gz/.whl on tag v*.
- Added bin/bump_version.py to bump versions and update pinned @vX.Y.Z references.
